## ğŸš€ Quick Start

### Prerequisites
- **Java 17+** for backend
- **Node.js 16+** for frontend
- **npm** or **yarn** package manager

### Backend Setup

1. **Navigate to backend directory:**
   ```bash
   cd backend
   ```

2. **Run the application:**
   ```bash
   ./gradlew bootRun
   ```
   
   Or build and run:
   ```bash
   ./gradlew build
   java -jar build/libs/printerserver-0.0.1-SNAPSHOT.jar
   ```

3. **Access the API:**
   - API Base URL: `http://localhost:8080`
   - OpenAPI Documentation: `http://localhost:8080/swagger-ui.html`

### Frontend Setup

1. **Navigate to frontend directory:**
   ```bash
   cd frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm start
   ```

4. **Access the application:**
   - Frontend URL: `http://localhost:3000`

## ğŸ“š API Documentation

### Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/files/all` | Get paginated file list |
| `GET` | `/files/{id}` | Get specific file details |
| `POST` | `/files/save` | Upload new file |
| `DELETE` | `/files/delete/{id}` | Delete file |
| `POST` | `/files/print/{id}` | Print file |

### Example Usage

```bash
# Get all files
curl -X GET "http://localhost:8080/files/all?page=0&size=20"

# Upload a file
curl -X POST "http://localhost:8080/files/save" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@document.pdf"

# Print a file
curl -X POST "http://localhost:8080/files/print/1"

# Delete a file
curl -X DELETE "http://localhost:8080/files/delete/1"
```

## ğŸ”§ Configuration

### Backend Configuration
Edit `application.properties` in `backend/src/main/resources/`:

```properties
# Server configuration
server.port=8080

# Database configuration
spring.datasource.url=jdbc:h2:mem:testdb
spring.datasource.driverClassName=org.h2.Driver

# File upload configuration
spring.servlet.multipart.max-file-size=10MB
spring.servlet.multipart.max-request-size=10MB
```

### Frontend Configuration
Edit `package.json` in `frontend/` to change backend proxy:

```json
{
  "proxy": "http://localhost:8080"
}
```
